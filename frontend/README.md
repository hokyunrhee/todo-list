## Assignment 2

- Todo 컴포넌트를 구현한다.
  - [ ] NewTask, Task List, Footer 컴포넌트를 조합한다.
  - [ ] 새로운 Task를 추가할 수 있다.
  - [ ] Task의 상태를 변경할 수 있다.
  - [ ] Task를 지울 수 있다.
  - [ ] Task title을 수정할 수 있다. 빈 title을 입력하면 해당 Task가 지워진다.
  - [ ] 모든 Task의 상태를 변경할 수 있다.
  - [ ] 완료된 모든 Task를 한번에 지울 수 있다.
  - [ ] 완료되지 않은 Task 개수를 보여준다.
  - [ ] all, active, completed 상태별로 Task를 필터링해서 볼 수 있다.
