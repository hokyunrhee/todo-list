# Todolist Backend

## Getting Started
아래 세 단계를 거치면 백엔드 코드 작성 준비가 된다.

### Installing Python
```
pip install pyenv
pyenv global 3.9.6
pyenv version
```

### Activating python virtual environment
```bash
cd python
python -m venv .venv
source .venv/bin/activate
```

### Installing Dependencies
```bash
pip install -r requirements.txt
```
